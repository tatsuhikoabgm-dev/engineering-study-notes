
# 学習用のリポジトリ

## 目的
これから必要になる技術について学習した内容をまとめる為に作りました
合わせて、マークダウン・マーメイドの記述方法、issueでの管理を学びます
今までUIでの操作に頼っていましたが、これを機にコマンド操作で頑張ります
gitbushを使います

## 学習キーワード (2026-2-17)

- EC2
- EKS
- Docker　*コマンド操作
- Kubernetes (K8s)
- JUnit
- karate
- Spring REST Docs
- サブネット　ルーティングテーブル
- PostgreSQL
- NginX

## 実践
- パブリックコンテナイメージ(Docker Hub)からNginXをpullしてローカルで起動ブラウザから見る

- パブリックサブネットからプライベートサブネットにSSHを使って接続ネットワーク構築

## 運用方法
- テンプレを使って調べたことをまとめていく
- 出典はリンクで貼ること
- 親issueの調査中に疑問に思ったことは調査のラベルを付けて子issueとして管理すること
- コミットは **#Issue番号**を付ける
